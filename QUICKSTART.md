# AdsClub - Deployment Quick Start

## 🚀 Deploy in 5 Minutes (Free)

### 1. Railway.app (Easiest)
```bash
# 1. Go to https://railway.app
# 2. Click "Start New Project"
# 3. Choose "Deploy from GitHub"
# 4. Select AdsClub7 repository
# 5. Add environment variables:
#    - MONGODB_URI (from MongoDB Atlas)
#    - JWT_SECRET (any random string)
# 6. Deploy!
```

**Your URL:** `https://your-app.railway.app`

### 2. Get Free Database (MongoDB Atlas)
```bash
# 1. Go to https://www.mongodb.com/cloud/atlas
# 2. Create free account
# 3. Create free cluster
# 4. Get connection string
# 5. Add to Railway environment variables
```

### 3. Optional: Custom Domain (Free)
```bash
# 1. Get free domain from freenom.com
# 2. Point DNS to Railway
# 3. Set in Railway custom domain settings
```
